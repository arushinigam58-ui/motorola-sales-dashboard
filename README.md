# 📊 Motorola Sales Dashboard

A professional Sales Analytics Dashboard built using **Power BI** and **Excel** to analyze Motorola mobile sales data across India.

## 📌 Project Overview

This dashboard provides a comprehensive view of Motorola's mobile sales performance across major Indian cities. It helps business stakeholders track sales trends, payment preferences, and customer behavior in one place.

---

## 🔍 Key Insights

- 📦 **Total Transactions:** 1.514K
- 💰 **Total Sales:** ₹308M
- 🛒 **Total Quantity Sold:** 7,622 units
- 📈 **Average Sales:** ₹41K

---

## 📊 Dashboard Features

- **City-wise Sales Map** — Visual representation of sales across Indian cities like Delhi, Mumbai, Kolkata, Chennai and more
- **Monthly Sales Trend** — Units sold tracked across all 12 months
- **Payment Method Breakdown** — Credit Card, UPI, Cash, Debit Card analysis
- **Brand Comparison Table** — Apple, OnePlus, Vivo and other brand performance
- **Customer Rating Analysis** — Rating distribution from 1 to 5
- **Day-wise Sales Performance** — Which days generate highest revenue
- **Interactive Filters** — Filter by Mobile Model, Payment Method, Brand and Day
